## declare

1. 函数定义: `fun NAME(VAR_NAME VAR_TYPE) : RETURN_TYPE`

   ```kotlin
   // 1.简单定义
   fun sum(a: Int, b: Int): Int {
       return a + b
   }

   // 2.等价于
   fun sum(a: Int, b: Int) = a + b
      fun sum(a: Int, b: Int): Int = a + b

   // 3.返回值类型为Unit可以省略
   fun printSum(a: Int, b: Int) = println("sum of $a and $b is ${a + b}")
   ```

2. 构造函数

   ```kotlin
   open class A { }

   class ConClass : A {
       constructor()
       constructor(s: String): this()
   }
   ```
